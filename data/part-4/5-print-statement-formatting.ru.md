К этому моменту мы познакомились с тремя способами сформировать аргументы, передаваемые команде `print`.

Первый — оператор `+` для строк. Он позволяет просто «склеивать» фрагменты строк:

```python
name = "Mark"
age = 37
print("Hi " + name + " your age is " + str(age) + " years" )
```

Этот способ не сработает, если хотя бы один фрагмент не является строкой. В примере выше переменная `age` была преобразована в строку с помощью `str`, так как это целое число и «склеить» его со строкой напрямую нельзя.

Второй способ — передавать каждую часть как отдельный аргумент, разделяя их запятыми:

```python
print("Hi", name, "your age is", age, "years" )
```

Этот код даёт ровно тот же результат, что и предыдущий вариант. Команда `print` по умолчанию добавляет пробел между аргументами. Преимущество здесь в том, что аргументы могут быть разных типов — ничего не нужно специально превращать в строку.

Если вы хотите убрать автоматически добавляемые пробелы, можно использовать именованный аргумент `sep`:

```python
print("Hi", name, "your age is", age, "years", sep="")
```

Это напечатает:

<sample-output>

HiMarkyour age is37years

</sample-output>


Аргумент `sep=""` — это _именованный аргумент_ (keyword argument). Его имя — сокращение от _separator_ (разделитель). Он задаёт строку, которой будут разделяться остальные аргументы. Разделителем может быть любая строка. Например, если нужно вывести каждый аргумент на отдельной строке, можно использовать `"\n"` — символ перевода строки:

```python
print("Hi", name, "your age is", age, "years", sep="\n")
```

<sample-output>

Hi
Mark
your age is
37
years

</sample-output>

По умолчанию `print` всегда заканчивает вывод переводом строки, но это тоже можно изменить. Именованный аргумент `end` задаёт, что будет добавлено в конце строки. Если установить `end` в пустую строку, перевода строки в конце не будет:

```python
print("Hi ", end="")
print("there!")
```

<sample-output>

Hi there!

</sample-output>

## f-строки {#f-strings}

Третий способ формирования строк — f-строки. Пример с именем и возрастом можно переписать так:

```python
name = "Erkki"
age = 39
print(f"Hi {name} your age is {age} years")
```

До сих пор мы использовали самые простые f-строки, но они позволяют гибко форматировать вывод. Один из частых случаев — управлять количеством знаков после запятой при выводе вещественных чисел. По умолчанию знаков довольно много:

```python
number = 1/3
print(f"The number is {number}")
```

<sample-output>

The number is 0.333333333333333

</sample-output>

Формат, в котором число должно отображаться, задаётся внутри фигурных скобок выражения. Добавим двоеточие и _спецификатор формата_ после имени переменной:

```python
number = 1/3
print(f"The number is {number:.2f}")
```

```python
The number is 0.33
```

Спецификатор `.2f` означает, что нужно показать 2 знака после запятой. Буква _f_ в конце говорит о том, что значение выводится как `float`, то есть как число с плавающей точкой.

Ещё один пример: зададим, сколько места (пробелов) зарезервировать под переменную в выводе. В обоих случаях для `name` выделено 15 символов. Сначала имена выравниваются по левому краю, а затем — по правому:

```python
names =  [ "Steve", "Jean", "Katherine", "Paul" ]
for name in names:
  print(f"{name:15} centre {name:>15}")
```

```python
Steve           centre           Steve
Jean            centre            Jean
Katherine       centre       Katherine
Paul            centre            Paul
```

f-строки полезны не только в `print`. Их можно присваивать переменным и соединять с другими строками:

```python
name = "Larry"
age = 48
city = "Palo Alto"
greeting = f"Hi {name}, you are {age} years of age"
print(greeting + f", and you live in {city}")
```

<sample-output>

Hi Larry, you are 48 years of age, and you live in Palo Alto

</sample-output>

Можно думать о f-строке как о «функции», которая создаёт обычную строку на основе «аргументов» внутри фигурных скобок.

<programming-exercise name='Числа в строки' anchor="Integers to strings" tmcname='part04-32_integers_to_strings'>

Напишите функцию `formatted`, которая принимает список вещественных чисел (тип `float`). Функция должна вернуть новый список, где каждый элемент исходного списка представлен строкой, округлённой до двух знаков после запятой. Порядок элементов должен остаться тем же, что и в исходном списке.

_Подсказка: используйте f-строки, чтобы преобразовать числа с плавающей точкой в строки нужного формата._

Пример ожидаемого поведения:

```python
my_list = [1.234, 0.3333, 0.11111, 3.446]
new_list = formatted(my_list)
print(new_list)
```

<sample-output>

['1.23', '0.33', '0.11', '3.45']

</sample-output>

</programming-exercise>
