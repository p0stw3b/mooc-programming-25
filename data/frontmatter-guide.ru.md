### Все возможные поля frontmatter {#kaikki-mahdolliset-frontmatter-fieldit}

banner

- Необязательное поле
- Тип: boolean
- Добавляет изображение `banner.svg` из корня проекта в верхнюю часть страницы.
- **Внимание:** баннер добавляется только если `information_page` тоже равен `true`.

hidden

- Необязательное поле
- Тип: boolean
- Страница не отображается в боковой панели (лучше использовать `hide_in_sidebar`, если вы хотите скрыть страницу только из боковой панели).
- При попытке открыть URL страницы будет 404.

hide\_in\_sidebar

- Необязательное поле
- Тип: boolean
- Скрывает страницу из боковой панели, если иначе она была бы там видна. По умолчанию в боковой панели отображаются все файлы с именем `index.md`, а также все markdown‑файлы в корне папки `data`.

information\_page

- Необязательное поле
- Тип: boolean
- Для боковой панели: страницы, у которых `information_page` равен `true`, показываются первыми.

overview

- Необязательное поле
- Тип: boolean
- Предназначено для индексных страниц частей курса (например `part-01/index.md`). Скрывает список подразделов части, который иначе выводится внизу страницы. **Внимание:** этот список также скрыт для страниц с `information_page`, поэтому на таких страницах не нужно отдельно выставлять `overview: false`.

path

- Обязательное поле
- Тип: string
- Путь страницы в URL в формате `baseurl/path`.

separator\_after

- Необязательное поле
- Тип: string
- Добавляет разделитель в боковой панели после ссылки на страницу.
- Если задано пустой строкой (`separator_after: ""`), будет показан разделитель без заголовка. Если строка не пустая, под разделителем будет заголовок с заданным текстом.

sidebar\_priority

- Необязательное поле
- Тип: integer
- Предназначено только для страниц с `information_page`.
- Позволяет переупорядочить ссылки на страницы `information_page` в боковой панели. По умолчанию сначала идут страницы `information_page`, затем остальные; и те и другие отсортированы по алфавиту.
- Страницы с полем `sidebar_priority` идут перед остальными и сортируются по убыванию: `sidebar_priority: 5000` будет раньше, чем `sidebar_priority: 2000`.

title

- Обязательное поле
- Тип: string
- Заголовок страницы: показывается в заголовке/вкладке браузера и вверху страницы.

upcoming

- Необязательное поле
- Тип: string
- Если задано, ссылка на страницу в боковой панели будет серой и некликабельной, а рядом будет показан указанный текст.
- Рекомендуем указывать дату публикации страницы в формате `дд.мм.`: поле достаточно маленькое, и год целиком не помещается.

vocabulary\_page

- Необязательное поле
- Тип: boolean
- Предназначено только для страницы глоссария. Если включено, на странице будет доступ к данным со словами глоссария.
